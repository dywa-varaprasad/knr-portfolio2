# 🌐 DVP Design – Business Website Template

A modern, mobile-first business website built for local businesses in Karimnagar, Telangana.

Designed for high conversion, speed, and professional branding.

---

## 🚀 Live Preview
👉 (https://dvpdesigns.netlify.app/)


---

## ✨ Features

✅ Fully Responsive (Mobile First)  
✅ Dark / Light Mode Toggle  
✅ GSAP Animations  
✅ WhatsApp Direct Contact Button  
✅ SEO Optimized Meta Tags  
✅ Sticky Navigation  
✅ Scroll Animations  
✅ Pricing & Portfolio Sections  
✅ High Conversion UI Design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Custom Variables + Dark Mode)  
- JavaScript (Vanilla JS)  
- GSAP Animation Library  
- Google Fonts  

No frameworks. No dependencies. Lightweight and fast.
